# rust-tutorial

官网：https://www.rust-lang.org/zh-CN/tools/install


### 环境：

macOS


### 开始
1. 安装

    ```shell
    curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
    ```

2. VSCode 安装插件 rust
3. 查看 rust 版本

    ```shell
    rustc -V
    ```

4. 创建工程

    ```shell
    cargo new project1
    ```
5. 编译与运行

    ```shell
    cargo build
    cargo run
    ```


### 常用

```shell
cargo doc # 生产文档 
```
